# MiddleEarth-Thaumaturgy
This is one of the minecraft mod that add contents themed LotR and Thaumcraft 4.

## Dependencies
- The Lord of the Rings mod v34.1 or above
- Baubles
- Thaumcraft 4
- TravellersGear(Optional)
- Automagy(Optional)
